Before you install

Windows will probably warn you.

New software has no reputation with Microsoft SmartScreen, so the first time you run the installer you may see a blue “Windows protected your PC” screen. Click More info, then Run anyway.

This is normal for small independent software and it fades as more people install. Every release here is signed and built in the open — you can read the build workflow, and the source, on GitHub.
